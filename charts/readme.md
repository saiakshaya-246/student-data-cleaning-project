# Student Data Cleaning & Visualization Project

## Objective

Clean raw student dataset and visualize meaningful insights using Python.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Data Cleaning

* Checked missing values
* Checked duplicate rows
* Verified dataset consistency
* Saved cleaned dataset

## Visualizations

1. Students by Department
2. GPA Distribution
3. Age vs GPA
4. Department-wise GPA Analysis

## Insights

* Dataset contains clean records with no missing values.
* GPA distribution shows average academic performance.
* Department comparison reveals GPA variation across fields.
* Student ages mostly fall within a narrow range.

## Outcome

Built a complete data cleaning and visualization workflow suitable for reporting and dashboard creation.
